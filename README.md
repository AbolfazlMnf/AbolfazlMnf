### 📖 The Story So Far...

My fascination with software didn't just start with writing syntax; it began with the realization that I could solve complex real-world problems through logic. During my Computer Engineering studies, I found my true calling in web development.

I started by mastering the visual web, diving deep into React. Building interfaces was fun, but managing data and state across components was the real puzzle. I spent my time mastering state management and data fetching to orchestrate smooth, flawless user flows.

As my applications grew, so did the potential for bugs. That's when I adopted **TypeScript**. Suddenly, my codebase was predictable, and catching errors before runtime felt like an absolute game-changer. 

But client-side apps weren't enough. I wanted my projects to be lightning-fast, highly secure, and easily discoverable by search engines. The turning point for me was discovering **Next.js**. Learning about Server-Side Rendering (SSR), Static Generation (SSG), and ISR completely shifted my perspective. I became deeply focused on Core Web Vitals, technical SEO, and delivering unmatched user experiences.

Beautiful frontends, however, need powerful engines. To gain full control over the systems I was building, I ventured into the backend. I immersed myself in **Node.js and Express.js**, designing secure REST APIs. I tackled database architecture, mastering both NoSQL (**MongoDB**) and relational databases (**PostgreSQL**), using tools like **Prisma ORM** to connect everything seamlessly.

Today, I build comprehensive digital solutions—from AI-integrated real estate platforms to complete e-commerce architectures. I’m constantly exploring new design patterns, optimizing query performance, and pushing the boundaries of what a modern web app can achieve.

If you're interested in building scalable products or just want to talk about clean code and architecture, let's connect!
